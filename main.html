How would you explain the concept of closures in JavaScript to a non-technical person?
Closures are functions that remember the environment in which they were created
Imagine you have a backpack. When you create a function, it gets a backpack that holds all the variables it can see around it at birth. Even when you take that function somewhere else, it still has its backpack with those original items.
Closures are a security feature that prevents access to certain variables
They're a type of function that automatically runs when the page loads
Hint: Think about how functions can "remember" values from their surrounding context even after that context is gone. A good analogy often helps explain technical concepts to non-technical people.
